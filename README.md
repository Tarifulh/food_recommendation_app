# Food Recommendation App

A Python-based food recommendation system that allows users to search recipes by recipe names or ingredients and view complete cooking information.

## Features

✅ Search recipes by name  
✅ Search recipes using ingredients  
✅ Displays country of origin  
✅ Shows main ingredients  
✅ Shows spices used  
✅ Displays cooking instructions  

## Technologies Used

- Python
- Dictionaries
- Lists
- Loops
- Conditional statements
- String handling

## Example Usage

Input:

```text
Search recipe / ingredients: chicken
```

Output:

```text
Recipe Found:
Chicken Masala Curry

Country:
India

Main Ingredients:
Chicken
Onion
Ginger and Garlic
```
